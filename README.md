# 🥊 UFC Fight Hub

**UFC Fight Hub** is a dynamic Flask web application that delivers fight predictions, fighter analysis, and custom bet slip generation for upcoming UFC events. Built for fans, analysts, and bettors alike, it combines data scraping, probabilistic modeling, and PDF generation into one sleek platform.

---

## 🚀 Features

- 🔮 **Fight Prediction Engine**  
  Simulate matchups and get win probabilities based on fighter stats and randomization logic.

- 📊 **Fighter Database**  
  View fighter records, weight classes, and profile images pulled from UFC and Wikipedia.

- 🧾 **PDF Bet Slip Generator**  
  Create and download a custom parlay bet slip with method-of-victory picks and QR codes.

- 🧠 **Fight Analysis**  
  Compare champions, contenders, and stylistic matchups with visual summaries.

- 🌐 **Responsive Frontend**  
  Clean HTML templates and CSS styling for a smooth user experience.

---

## 🛠 Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS (custom)  
- **Database:** SQLite  
- **PDF & QR Code:** ReportLab, qrcode  
- **Web Scraping:** BeautifulSoup, Requests  
- **Version Control:** Git + GitHub
